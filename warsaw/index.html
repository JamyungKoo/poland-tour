Lecture 06 · Mobile Navi

바르샤바 가이드 투어

중앙역 → 게토 영웅 추모비 · 도보 6km · 13정거장. 모바일에서 보기 편한 두 가지 버전을 준비했습니다.

v3 · 유연한 동선 추천

v1과 v2의 장점을 합쳐 — 인터랙티브 지도 + 구글 임베드를 모두 제공. 드래그로 순서 변경, Pass로 정거장 건너뛰기가 가능하고, 변경 즉시 임베드 경로가 자동 갱신됩니다.

  • 상단 인터랙티브 지도 (사용자 순서 반영)
  • 도보 구간 = 구글 임베드 (자동 갱신)
  • ⠿ 길게 눌러 드래그 = 순서 변경
  • Pass = 동선에서 제외 (회색 표시)
  • 맨 위로 버튼 + 순서 초기화

v1 · 인터랙티브 지도 (고정 동선)

전체 루트가 표시된 OSM 인터랙티브 지도 + 구글지도 길찾기 딥링크. 원본 동선 그대로, 순서 변경 없이 가장 가볍게.

  • 전체 루트 한 눈에 확인 (핀 13개 + 경로선)
  • 현재 위치 표시 + 가장 가까운 정거장 안내
  • 가장 가벼움 (이미지 외 외부 임베드 없음)
  • 안드로이드: 구글지도 앱 자동 호출

v2 · 구글 지도 임베드 (고정 동선)

각 도보 구간마다 구글 지도 미리보기 iframe이 삽입되어 있습니다. 페이지 안에서 바로 경로를 확인할 수 있지만, 페이지가 무겁고 데이터 사용량이 큽니다.

  • 구간마다 구글 지도 미리보기 (12개 iframe)
  • 오프라인에서는 지도 미표시
  • 인터랙티브 지도 없음
  • 데이터/배터리 사용 많음
💡 사용 팁
  • 홈 화면에 추가 — 안드로이드 Chrome 메뉴 → "홈 화면에 추가"
  • 방문 체크 — 각 정거장 "방문" 버튼은 자동 저장됩니다 (브라우저 데이터)
  • 길찾기 — "구글 지도 길찾기" 버튼은 이전 정거장 → 현재 정거장 도보 경로를 띄웁니다
  • 현재 위치 — v1에서 "📍 내 위치" 버튼을 누르면 권한 요청 후 표시
